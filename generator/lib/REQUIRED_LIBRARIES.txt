Please copy the following list of libraries from a JChem Suite cross platform ZIP package into this directory.

The libraries should be contained in the jchem/lib/ subdirectory of the extracted ZIP file.

The application has been verified to work with version 16.1.4.0, although it might be possible to use a later version.
